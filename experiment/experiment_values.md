# Guide to Experiment Values

Here is documented the values used for each variable in the experimental framework, for clarity.

* **Duration (s):** 10, 20, 30
* **Parallelism Level:** 1, 2, 4, 8, 16, 32, 64
* **Badwidth cap (bitrate, Gbps):** 25, 50, 75, 100, 150, 200
* **Protocol:** tcp, udp, quic
* **Delay (ms):** 10, 100, 1000
* **Loss (%):** 0, 25, 50, 75, 99
